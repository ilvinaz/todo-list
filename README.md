# To-do List App
A simple and stylish to-do list application built with HTML, CSS and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Smooth animations and modern UI
- Responsive layout

## Technologies
- HTML5
- CSS3 (flexbox, animations)
- JavaScript (DOM manipulation)
